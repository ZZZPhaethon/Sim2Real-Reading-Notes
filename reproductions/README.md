# Reproductions

The repository currently includes two validated experiments:

| Experiment | Scope | Hardware | Result |
|---|---|---|---|
| [PAct](pact/README.md) | Single-image articulated-object inference smoke test | NVIDIA A100-SXM4-40GB | Completed end to end |
| [TRELLIS.2 O-Voxel](ovoxel/README.md) | Textured mesh to O-Voxel conversion and visualization | NVIDIA A100-SXM4-40GB | Completed at `64^3` and `256^3` |

These records distinguish executable smoke tests from full paper reproduction.
Pretrained weights, upstream repositories, input datasets, and virtual
environments are intentionally excluded.
